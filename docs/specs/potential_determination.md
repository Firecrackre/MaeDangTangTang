## 1. 개요

**잠재능력 판정**은 플레이어가 큐브를 사용했을 때, 아이템에 부여될 잠재능력 3줄의 결과를 생성하고 판정하는 규칙이다.

큐브를 사용하면 먼저 잠재능력 등급이 결정되고, 이후 3개의 옵션이 생성된다.

각 옵션은 **주요 옵션 / 잡옵 / 파괴** 중 하나로 분류되며, 일부 주요 옵션은 낮은 확률로 일반 수치보다 높은 **이탈 옵션**으로 등장할 수 있다.

생성된 결과는 즉시 플레이어에게 공개되지 않고, **큐브 스크래치 화면**으로 전달되어 은박으로 가려진 상태로 표시된다.

---

## 2. 목표

잠재능력 판정의 목표는 다음과 같다.

| 목표 | 설명 |
| --- | --- |
| 결과 다양성 제공 | 큐브 사용마다 서로 다른 잠재능력 결과가 생성되도록 한다. |
| 기대감 형성 | 주요 옵션, 이탈 옵션, 높은 등급을 통해 대박 가능성을 만든다. |
| 리스크 부여 | 파괴 항목을 통해 큐브 사용에 위험 요소를 부여한다. |
| 판매가 연동 | 옵션 결과가 아이템 예상 판매가에 반영되도록 한다. |
| 스크래치 연동 | 생성된 결과를 큐브 스크래치 화면에서 한 줄씩 공개할 수 있도록 전달한다. |

---

## 3. 판정 흐름

큐브 사용 시 잠재능력 판정은 다음 순서로 진행된다.

```
1. 플레이어가 큐브를 사용한다.
2. 큐브 종류를 확인한다.
3. 잠재능력 등급을 결정한다.
4. 옵션 1을 생성한다.
5. 옵션 2를 생성한다.
6. 옵션 3을 생성한다.
7. 각 옵션의 타입을 판정한다.
8. 주요 옵션의 경우 이탈 여부를 판정한다.
9. 3줄 중 파괴 항목 포함 여부를 확인한다.
10. 3줄 조합 결과를 판정한다.
11. 결과 데이터를 큐브 스크래치 화면으로 전달한다.
```

---

## 4. 등급

잠재능력 등급은 큐브 사용 시 가장 먼저 결정된다.

등급은 아이템의 최종 가치에 영향을 주며, 등급이 높을수록 좋은 결과가 나왔을 때 더 높은 판매가를 기대할 수 있다.

| 등급 | 설명 | 판매가 배수 |
| --- | --- | --- |
| 레어 | 가장 기본적인 잠재능력 등급 | x1.0 |
| 에픽 | 기본보다 높은 가치를 기대할 수 있는 등급 | x2.0 |
| 유니크 | 좋은 옵션 조합이 나왔을 때 높은 수익을 기대할 수 있는 등급 | x3.5 |
| 레전더리 | 가장 높은 등급으로, 대박 결과의 가치가 크게 상승하는 등급 | x5.0 |

### 등급 확률 기준안

| 등급 | 등장 확률 |
| --- | --- |
| 레어 | 60% |
| 에픽 | 28% |
| 유니크 | 10% |
| 레전더리 | 2% |

> 등급 확률은 초기 밸런싱 기준안이며, 실제 플레이 테스트 결과에 따라 조정한다.
>

## 5. 옵션 타입

각 잠재능력 줄은 다음 세 가지 타입 중 하나로 생성된다.

| 타입 | 설명 |
| --- | --- |
| 주요 옵션 | 판매가를 올리는 핵심 옵션 |
| 잡옵 | 판매가 상승폭이 거의 없는 일반 옵션 |
| 파괴 | 등장 시 아이템을 파괴시키는 위험 항목 |

### 옵션 타입 확률 기준안

| 옵션 타입 | 등장 확률 |
| --- | --- |
| 주요 옵션 | 45% |
| 잡옵 | 45% |
| 파괴 | 10% |

> 옵션 타입 확률은 각 줄마다 개별 판정한다.
> 
> 
> 단, 장인의 큐브처럼 파괴가 없는 큐브가 존재할 경우 파괴 확률은 0%로 처리한다.
>

## 6. 주요 옵션

주요 옵션은 아이템 판매가를 크게 상승시키는 옵션이다.

주요 옵션은 크게 **주스탯 옵션**과 **공격 계열 옵션**으로 나뉜다.

| 분류 | 옵션 |
| --- | --- |
| 주스탯 | STR, DEX, INT, LUK |
| 공격 계열 | 공격력, 마력, 보스 공격력 |

### 주스탯 옵션

| 옵션 | 일반 수치 | 이탈 수치 |
| --- | --- | --- |
| STR | +9% | +12% |
| DEX | +9% | +12% |
| INT | +9% | +12% |
| LUK | +9% | +12% |

### 공격 계열 옵션

| 옵션 | 일반 수치 | 이탈 수치 |
| --- | --- | --- |
| 공격력 | +6% | +9% |
| 마력 | +6% | +9% |
| 보스 공격력 | +20% | +30% |

### 주요 옵션 생성 기준

| 항목 | 내용 |
| --- | --- |
| 중복 허용 | 동일 옵션 중복 가능 |
| 동일 주스탯 | 같은 주스탯이 여러 줄 등장할 수 있음 |
| 공격 계열 조합 | 공격력, 마력, 보스 공격력이 함께 등장할 수 있음 |
| 이탈 판정 | 주요 옵션에 한해 이탈 여부를 추가 판정 |

---

## 7. 잡옵

잡옵은 판매가 상승폭이 매우 낮은 일반 옵션이다.
잡옵은 가치가 낮지만, 파괴처럼 아이템을 잃게 만들지는 않는다.

| 옵션 | 수치 예시 |
| --- | --- |
| 수면 효율 | +30% |
| 이동속도 | +10 |
| 점프력 | +10 |
| 언어 능력 | +30% |
| 친화력 | +30% |

### 잡옵 처리 기준

| 항목 | 내용 |
| --- | --- |
| 판매가 영향 | 낮은 판매가 보너스 적용 |
| 이탈 여부 | 이탈 옵션 없음 |
| 조합 보너스 | 없음 |
| 공개 피드백 | 기본 공개 효과 또는 꽝 효과음 |

---

## 8. 이탈 옵션

이탈 옵션은 주요 옵션이 일반 수치보다 높은 값으로 등장하는 경우를 의미한다.

이탈 옵션은 낮은 확률로 등장하며, 등장 시 아이템의 예상 판매가를 크게 상승시킨다.

| 항목 | 내용 |
| --- | --- |
| 발생 대상 | 주요 옵션 |
| 발생 방식 | 주요 옵션 생성 후 추가 확률 판정 |
| 효과 | 일반 수치보다 높은 옵션 수치 적용 |
| 판매가 영향 | 이탈 배수 또는 추가 보너스 적용 |
| 피드백 | 강한 반짝임, 특수 효과음 |

### 이탈 확률 기준안

| 조건 | 이탈 확률 |
| --- | --- |
| 주요 옵션 생성 시 | 15% |

### 이탈 개수별 가치 기준

| 이탈 개수 | 가치 기준 |
| --- | --- |
| 0개 | 일반 결과 |
| 1개 | 좋은 결과 |
| 2개 | 매우 좋은 결과 |
| 3개 | 초대박 결과 |

---

## 9. 파괴 항목

파괴 항목은 잠재능력 3줄 중 하나로 등장할 수 있는 위험 항목이다.

3줄 중 하나라도 파괴 항목이 포함되면, 해당 아이템은 최종적으로 파괴 처리된다.

| 항목 | 내용 |
| --- | --- |
| 발생 위치 | 옵션 1 / 옵션 2 / 옵션 3 중 하나 |
| 발생 조건 | 옵션 타입 판정에서 파괴 선택 |
| 중복 가능 여부 | 가능 |
| 최종 판정 | 파괴 항목이 1개 이상이면 파괴 |
| 우선순위 | 모든 좋은 옵션, 이탈 옵션, 등급보다 우선 |

### 처리 기준

파괴 항목은 다른 모든 판정보다 우선한다.

따라서 좋은 옵션이나 이탈 옵션이 함께 등장하더라도, 파괴 항목이 1개 이상 포함되어 있다면 최종 결과는 파괴로 처리한다.

예시:

```
등급: 레전더리

옵션 1: STR +12%
옵션 2: 보스 공격력 +30%
옵션 3: 파괴
```

위 결과는 레전더리 등급과 이탈 옵션 2개가 포함되어 있지만, 파괴 항목이 존재하므로 최종 결과는 파괴로 판정한다.

---

## 10. 조합 판정

잠재능력 3줄이 모두 생성되면, 3줄 조합을 기준으로 결과를 판정한다.

| 결과 | 판정 기준 |
| --- | --- |
| 파괴 | 파괴 항목 1개 이상 포함 |
| 신급 대박 | 공격 계열 3줄 + 이탈 3개 |
| 전설급 대박 | 동일 주스탯 3줄 + 이탈 3개 |
| 특급 대박 | 주요 옵션 3줄 + 이탈 포함 |
| 공격 대박 | 공격 계열 3줄 |
| 주스탯 대박 | 동일 주스탯 3줄 |
| 대박 | 주요 옵션 3줄 |
| 성공 이상 | 주요 옵션 2줄 + 이탈 포함 |
| 성공 | 주요 옵션 2줄 |
| 보통 | 주요 옵션 1줄 |
| 꽝 | 잡옵 위주 |

### 판정 우선순위

조합 판정은 다음 우선순위를 따른다.

```
1. 파괴 여부
2. 공격 계열 3줄 + 이탈 3개
3. 동일 주스탯 3줄 + 이탈 3개
4. 주요 옵션 3줄 + 이탈 포함
5. 공격 계열 3줄
6. 동일 주스탯 3줄
7. 주요 옵션 3줄
8. 주요 옵션 2줄 + 이탈 포함
9. 주요 옵션 2줄
10. 주요 옵션 1줄
11. 잡옵 위주
```

## 11. 결과 데이터

잠재능력 판정이 완료되면, 다음 데이터를 큐브 스크래치 화면으로 전달한다.

| 데이터 | 설명 |
| --- | --- |
| 등급 | 레어 / 에픽 / 유니크 / 레전더리 |
| 옵션 1 | 첫 번째 잠재능력 정보 |
| 옵션 2 | 두 번째 잠재능력 정보 |
| 옵션 3 | 세 번째 잠재능력 정보 |
| 옵션 타입 | 주요 옵션 / 잡옵 / 파괴 |
| 이탈 여부 | 일반 / 이탈 |
| 파괴 여부 | true / false |
| 주요 옵션 개수 | 0~3 |
| 이탈 옵션 개수 | 0~3 |
| 조합 판정 | 꽝 / 보통 / 성공 / 대박 / 파괴 등 |
| 판매가 계산용 값 | 판매가 계산에 필요한 옵션 정보 |

### 옵션 데이터 구조 예시

```
OptionData
- optionName: STR
- optionType: 주요 옵션
- optionGroup: 주스탯
- value: 12%
- isBonusValue: true
- isDestroy: false
```

### 결과 데이터 구조 예시

```
PotentialResultData
- grade: Legendary
- option1: STR +12%
- option2: STR +9%
- option3: STR +9%
- hasDestroy: false
- mainOptionCount: 3
- bonusOptionCount: 1
- resultType: 주스탯 대박
```

## 12. 큐브 종류별 차이

큐브 종류에 따라 잠재능력 판정 규칙이 달라질 수 있다.

메커톤 버전에서는 큐브 종류를 과하게 늘리지 않고, 기본적으로 **이상한 큐브**를 중심으로 구현한다.

| 큐브 | 특징 |
| --- | --- |
| 이상한 큐브 | 기본 큐브. 낮은 비용으로 사용 가능하지만 파괴 항목이 등장할 수 있다. |
| 장인의 큐브 | 높은 비용이지만 파괴 항목이 등장하지 않는다. |

### 큐브별 판정 차이 기준안

| 항목 | 이상한 큐브 | 장인의 큐브 |
| --- | --- | --- |
| 파괴 항목 | 등장 가능 | 등장하지 않음 |
| 주요 옵션 확률 | 기본 | 기본 또는 소폭 증가 |
| 이탈 옵션 확률 | 기본 | 기본 또는 소폭 증가 |
| 등급 확률 | 기본 | 기본 또는 소폭 증가 |

> 장인의 큐브는 후순위 구현 대상으로 둔다.
메커톤 버전에서는 이상한 큐브만으로 핵심 루프를 검증하는 것을 우선한다.
>

## 13. 외부 연동

잠재능력 판정은 다른 기능과 연결되지만, 해당 기능의 세부 처리는 직접 관리하지 않는다.

| 연동 대상 | 처리 방식 |
| --- | --- |
| 큐브 | 큐브 사용 이벤트를 받아 판정을 시작한다. |
| 아이템 | 현재 큐브를 사용할 대상 아이템 정보를 참조한다. |
| 큐브 스크래치 | 생성된 잠재능력 3줄 결과를 전달한다. |
| 판매가 계산 | 옵션 정보, 등급, 이탈 여부, 조합 판정 값을 전달한다. |
| 파괴 처리 | 파괴 여부 값을 전달한다. |
| 연출 | 조합 판정과 옵션 타입 값을 전달한다. |
